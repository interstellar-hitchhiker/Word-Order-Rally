# Classroom Guide

Word Order Rally is built around two different lesson flows.

## Class Display

Use this when the game is projected at the front of the room.

1. Choose `Class Display`.
2. Choose CEFR, MYP, TEFL, or Teacher Bank.
3. Start the rally.
4. Students write or say the unscrambled sentence in pairs, teams, or mini-whiteboards.
5. When the timer reaches zero, the answer appears automatically.
6. Briefly discuss the word order and launch the next sentence.

This mode intentionally removes the answer tray and check button. A class display round should not make one person click words on behalf of everyone.

## Solo Practice

Use this when one learner or pair is working directly on the device.

1. Choose `Solo Practice`.
2. Learners tap words into the answer tray.
3. `Check answer` marks words green when they are in the right place and red when they are not.
4. The correct sentence appears underneath automatically.
5. Learners compare their version with the answer before moving on.

## Quick uses

- Warmers
- Grammar noticing
- Retrieval practice
- Exit tickets
- Pair races
- Mini-whiteboard review
- End-of-unit language focus
