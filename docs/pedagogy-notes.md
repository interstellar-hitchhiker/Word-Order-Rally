# Pedagogy Notes

Word Order Rally is a short sentence-level practice tool. Its value comes from reconstruction, comparison, and feedback rather than from the timer alone.

## Why two modes?

In class display, the device is a shared prompt. Students need time to think, write, discuss, and then compare with the reveal. A clickable answer tray is unnecessary there.

In solo practice, the device is the learner's workspace. Clicking words, checking placement, and seeing red/green feedback creates useful noticing before the correct answer is revealed.

## Level labels

The CEFR, MYP, and TEFL routes are practical classroom selectors, not formal assessment decisions. Teachers should import or write their own sentences when a course has specific topics, grammar, or language background.
